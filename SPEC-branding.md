# Robot Branding - Fichier de Tâches

Ce fichier définit les tâches d'analyse de branding inspirées des fonctionnalités de Branding5 pour évaluer et optimiser l'identité d'une marque.

---

## Principes d'Approche

### Regard Critique et Analytique
Toute analyse de branding doit être menée avec un regard critique sur ce qui est présenté et ce dont on parle. Ne jamais prendre les éléments de marque pour acquis : questionner la cohérence, la pertinence et l'authenticité de chaque composante. Croiser les données, identifier les biais, et distinguer ce qui relève du discours de marque de la réalité terrain.

### Explication des Méthodologies
Chaque framework, matrice ou outil d'analyse utilisé doit être explicitement expliqué : son origine, son fonctionnement, ses limites et dans quel contexte il est le plus pertinent. L'objectif est que le lecteur comprenne non seulement les résultats, mais aussi pourquoi et comment on y arrive.

### Dimension Éducationnelle
Ce document porte une forte dimension éducationnelle. Les multiples points de vue (archétypes de Jung, SWOT, AIDA, analyse psychographique, etc.) sont autant de grilles de lecture complémentaires. L'enjeu est de former un regard pluridisciplinaire sur le branding, pas seulement de produire un livrable. Chaque phase est une opportunité d'apprentissage et de montée en compétence.

## Phase 1 : Collecte des Informations de Base

### Tâche 1.1 : Identification de la Marque
- Récupérer le nom officiel de la marque et ses variantes
- Documenter l'URL du site web principal et les domaines secondaires
- Identifier le secteur d'activité et le marché cible
- Noter la date de création et les jalons historiques importants

### Tâche 1.2 : Mission, Vision et Valeurs
- Extraire la mission déclarée (raison d'être de la marque)
- Identifier la vision (objectif à long terme)
- Lister les valeurs fondamentales communiquées
- Évaluer la clarté et l'authenticité de ces éléments

### Tâche 1.3 : Inventaire des Éléments Visuels
- Collecter le logo principal et ses déclinaisons
- Documenter la palette de couleurs (codes HEX/RGB)
- Identifier la typographie utilisée (titres, corps de texte)
- Recenser les éléments graphiques récurrents (icônes, motifs, illustrations)

### Tâche 1.4 : Positionnement Actuel
- Analyser le tagline ou slogan actuel
- Identifier la proposition de valeur unique (UVP)
- Documenter le segment de prix (entrée de gamme, premium, luxe)
- Évaluer la perception actuelle sur le marché

## Phase 2 : Analyse des Archétypes de Marque (Jung)

### Tâche 2.1 : Identification de l'Archétype Dominant
Analyser la marque selon les 12 archétypes de Jung :

| Groupe | Archétypes |
|--------|------------|
| Stabilité | Le Souverain, Le Créateur, Le Soignant |
| Indépendance | L'Innocent, Le Sage, L'Explorateur |
| Maîtrise | Le Héros, Le Magicien, Le Rebelle |
| Appartenance | L'Amoureux, Le Bouffon, Le Compagnon |

- Identifier l'archétype principal de la marque
- Noter les archétypes secondaires éventuels
- Justifier le choix avec des exemples concrets du messaging

### Tâche 2.2 : Cohérence Archétype-Messaging
- Vérifier l'alignement entre l'archétype et le ton de communication
- Analyser les visuels par rapport aux codes de l'archétype
- Évaluer la cohérence sur les différents points de contact
- Identifier les incohérences ou contradictions

### Tâche 2.3 : Benchmark Archétypal
- Identifier 3 marques leaders utilisant le même archétype
- Analyser leurs stratégies de communication
- Documenter les bonnes pratiques observées
- Identifier les opportunités de différenciation

### Tâche 2.4 : Alignement Archétype-Audience
- Vérifier la résonance de l'archétype avec l'audience cible
- Analyser les attentes émotionnelles du public
- Évaluer si l'archétype inspire confiance et connexion
- Recommander des ajustements si nécessaire

## Phase 3 : Profil Client Idéal (ICP)

### Tâche 3.1 : Analyse Démographique
- Définir la tranche d'âge cible
- Identifier la localisation géographique prioritaire
- Estimer le niveau de revenus et pouvoir d'achat
- Documenter le niveau d'éducation et la profession type

### Tâche 3.2 : Analyse Psychographique
- Identifier les valeurs personnelles de l'audience
- Documenter les aspirations et objectifs de vie
- Lister les frustrations et points de douleur
- Comprendre les motivations d'achat profondes

### Tâche 3.3 : Analyse Comportementale
- Cartographier le parcours d'achat typique
- Identifier les canaux de communication préférés
- Documenter les habitudes de consommation de contenu
- Analyser les déclencheurs de décision d'achat

### Tâche 3.4 : Création de Personas
- Créer 2-3 personas détaillés représentatifs
- Donner un nom et une photo type à chaque persona
- Rédiger une journée type pour chaque persona
- Définir les messages clés qui résonnent avec chaque profil

---

## Phase 4 : Analyse SWOT

### Tâche 4.1 : Forces (Strengths)
- Identifier les avantages concurrentiels uniques
- Documenter les ressources et compétences distinctives
- Lister les actifs de marque (réputation, brevets, communauté)
- Évaluer la solidité des relations clients existantes

### Tâche 4.2 : Faiblesses (Weaknesses)
- Identifier les lacunes dans l'offre ou le service
- Documenter les limites de ressources (budget, équipe)
- Analyser les points faibles de la présence digitale
- Évaluer les incohérences dans le branding actuel

### Tâche 4.3 : Opportunités (Opportunities)
- Identifier les tendances de marché favorables
- Documenter les segments de clientèle inexploités
- Analyser les nouvelles technologies ou canaux émergents
- Évaluer les partenariats stratégiques potentiels

### Tâche 4.4 : Menaces (Threats)
- Identifier les concurrents en croissance
- Analyser les évolutions réglementaires impactantes
- Documenter les changements de comportement consommateur
- Évaluer les risques économiques ou sectoriels

---

## Phase 5 : Intelligence Concurrentielle

### Tâche 5.1 : Identification des Concurrents
- Lister 3-5 concurrents directs (même offre, même cible)
- Identifier 2-3 concurrents indirects (offre substituable)
- Documenter les nouveaux entrants potentiels
- Classifier par niveau de menace (faible, moyen, élevé)

### Tâche 5.2 : Analyse du Positionnement Concurrent
- Documenter la proposition de valeur de chaque concurrent
- Analyser leur stratégie de prix
- Évaluer leur présence et réputation digitale
- Identifier leurs forces et faiblesses perçues

### Tâche 5.3 : Matrice de Positionnement
- Définir 2 axes de différenciation pertinents
- Positionner la marque et ses concurrents sur la matrice
- Identifier les espaces de positionnement disponibles
- Recommander un positionnement optimal

### Tâche 5.4 : Différenciateurs Uniques
- Identifier ce que la marque fait mieux que tous les concurrents
- Documenter les éléments non-copiables (histoire, expertise, communauté)
- Formuler des angles de différenciation exploitables
- Prioriser les différenciateurs par impact potentiel

---

## Phase 6 : Audit Digital et SEO

### Tâche 6.1 : Audit des Landing Pages
- Évaluer la clarté de la proposition de valeur au-dessus de la ligne de flottaison
- Analyser la hiérarchie visuelle et le parcours de lecture
- Vérifier la présence et l'efficacité des appels à l'action (CTA)
- Tester la cohérence du message sur mobile et desktop

### Tâche 6.2 : Analyse SEO
- Identifier les mots-clés principaux ciblés
- Analyser les positions actuelles sur les requêtes stratégiques
- Évaluer la qualité du contenu et l'optimisation on-page
- Vérifier la structure technique (balises, URLs, sitemap)

### Tâche 6.3 : Performance Technique
- Mesurer le score Lighthouse (Performance, Accessibilité, SEO)
- Analyser les Core Web Vitals (LCP, FID, CLS)
- Identifier les problèmes de vitesse de chargement
- Documenter les erreurs techniques à corriger

### Tâche 6.4 : Cohérence Multi-Canal
- Auditer la présence sur les réseaux sociaux
- Vérifier l'uniformité visuelle sur tous les canaux
- Analyser la cohérence du messaging cross-platform
- Évaluer l'expérience utilisateur globale de la marque

---

## Phase 7 : Voix et Personnalité de Marque

### Tâche 7.1 : Définition du Ton de Voix
- Positionner sur l'axe formel ↔ informel
- Positionner sur l'axe expert ↔ accessible
- Positionner sur l'axe sérieux ↔ ludique
- Positionner sur l'axe autoritaire ↔ collaboratif

### Tâche 7.2 : Traits de Personnalité
- Identifier 5 adjectifs définissant la personnalité de marque
- Documenter ce que la marque est et ce qu'elle n'est pas
- Créer des exemples de phrases "on dit" vs "on ne dit pas"
- Valider l'alignement avec l'archétype identifié

### Tâche 7.3 : Guide de Style de Communication
- Définir les règles de vocabulaire (mots à utiliser, mots à éviter)
- Documenter les conventions grammaticales (tu/vous, majuscules)
- Créer des templates de messages types (email, social, pub)
- Établir les règles d'utilisation des émojis et ponctuation

### Tâche 7.4 : Validation du Messaging Existant
- Auditer 10-20 communications récentes de la marque
- Évaluer la cohérence avec le guide de style défini
- Identifier les écarts et incohérences
- Recommander des corrections prioritaires

---

## Phase 8 : Recommandations Stratégiques

### Tâche 8.1 : Synthèse des Insights
- Résumer les 5 découvertes clés de l'analyse
- Identifier les 3 problèmes prioritaires à résoudre
- Documenter les 3 opportunités majeures à saisir
- Créer un score global de santé de marque (1-10)

### Tâche 8.2 : Plan d'Action Priorisé
- Définir les actions court terme (0-3 mois)
- Planifier les initiatives moyen terme (3-6 mois)
- Esquisser la vision long terme (6-12 mois)
- Estimer les ressources nécessaires pour chaque phase

### Tâche 8.3 : Frameworks de Messaging
Appliquer le framework AIDA :
- **Attention** : Accroches et headlines recommandés
- **Intérêt** : Points de différenciation à mettre en avant
- **Désir** : Bénéfices émotionnels à communiquer
- **Action** : Appels à l'action optimisés

### Tâche 8.4 : Piliers de Contenu
- Définir 3-5 thématiques de contenu principales
- Documenter les formats de contenu prioritaires par canal
- Créer un calendrier éditorial type
- Établir les KPIs de performance par pilier

## Métriques de Succès

- Clarté du positionnement : score de différenciation perçue
- Cohérence de marque : pourcentage d'alignement cross-canal
- Résonance avec l'audience : taux d'engagement et sentiment
- Performance digitale : scores Lighthouse et Core Web Vitals
- Compétitivité : part de voix vs concurrents
